Primeros pasos

Esta documentación proporciona toda la información necesaria para integrar las APIs de eqhuma en sus aplicaciones. Aprenda cómo autenticarse, manejar errores, y optimizar su integración para obtener el mejor rendimiento.

Importante

Todas las solicitudes a la API deben incluir su API Key para autenticación. Asegúrese de mantener su API Key segura y no compartirla con terceros.

Estructura de la API

Nuestras APIs siguen una estructura RESTful con los siguientes puntos importantes:

  • La URL base para todas las APIs es https://api.eqhuma.mx/v1/
  • Todas las respuestas están en formato JSON
  • Los códigos de estado HTTP estándar se utilizan para indicar el éxito o fracaso de una solicitud
  • Se requiere autenticación para todas las solicitudes

Autenticación

La autenticación se realiza utilizando API Keys. Todas las solicitudes deben incluir su API Key como parte de los parámetros de la solicitud o como un encabezado HTTP.

Ejemplo de autenticación por parámetro

// Envío como parámetro en el cuerpo JSON { "curp": "OEGH901122HDFLLC09", "api_key": "tu_api_key_aqui" }

Ejemplo de autenticación por encabezado

// Envío como encabezado HTTP GET /v1/sat/rfc-by-curp Host: api.eqhuma.mx X-API-Key: tu_api_key_aqui Content-Type: application/json

Manejo de Errores

Las respuestas de error siguen un formato consistente que incluye un código de error, un mensaje descriptivo y, cuando corresponde, detalles adicionales para ayudar a diagnosticar el problema.

{ "success": false, "error": "invalid_parameter", "message": "El parámetro 'curp' tiene un formato inválido", "details": { "parameter": "curp", "provided": "ABCD", "reason": "Debe contener 18 caracteres" }, "timestamp": "2023-10-25T15:31:10Z" }

Rate Limiting

Para garantizar la estabilidad y disponibilidad del servicio, implementamos límites de velocidad en las solicitudes a nuestras APIs. Los límites pueden variar según el plan de suscripción.

Plan Límite de Solicitudes Período
Básico 100 solicitudes Por minuto
Profesional 500 solicitudes Por minuto
Empresarial 2,000 solicitudes Por minuto
