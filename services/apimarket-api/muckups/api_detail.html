Descripción General

Esta API permite obtener el RFC (Registro Federal de Contribuyentes) oficial registrado en el SAT (Servicio de Administración Tributaria) a partir de la CURP (Clave Única de Registro de Población) proporcionada. Es importante destacar que no se mostrarán resultados de RFC sin registro.

Endpoint

POST https://api.eqhuma.mx/v1/sat/rfc-by-curp

Parámetros de Solicitud

Parámetro Tipo Requerido Descripción
curp String Sí CURP de la persona a consultar (18 caracteres)
api_key String Sí Clave de API para autenticación

Ejemplo de Solicitud

{ "curp": "OEGH901122HDFLLC09", "api_key": "tu_api_key_aqui" }

Respuesta de Éxito

{ "success": true, "rfc": "OEGH901122ABC", "status": "active", "source": "sat", "message": "RFC encontrado", "timestamp": "2023-10-25T15:30:45Z" }

Respuesta de Error

{ "success": false, "error": "curp_not_found", "message": "No se encontró un RFC asociado a la CURP proporcionada", "timestamp": "2023-10-25T15:31:10Z" }

Códigos de Estado

Código Descripción
200 Éxito. La solicitud se procesó correctamente.
400 Solicitud incorrecta. Falta un parámetro requerido o la CURP tiene formato inválido.
401 No autorizado. La API key es inválida o ha expirado.
404 No encontrado. No se encontró un RFC asociado a la CURP proporcionada.
429 Demasiadas solicitudes. Has excedido el límite de solicitudes permitidas.
500 Error interno del servidor.
